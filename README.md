# Zepto Sales Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes Zepto sales performance using various KPIs and visualizations.

## Features
- Total Sales Analysis
- Average Sales
- Number of Items Sold
- Average Rating
- Outlet Size Analysis
- Outlet Location Analysis
- Item Type Performance
- Outlet Establishment Trends

## Tools Used
- Power BI
- Excel
- Data Visualization
